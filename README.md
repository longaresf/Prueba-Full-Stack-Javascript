# README.md

## Description
This repository contains a full-stack JavaScript application. It includes front-end and back-end components built using modern web technologies.

## Tech Stack
- **Frontend:** React.js, Redux for state management, Axios for HTTP requests.
- **Backend:** Node.js with Express framework, MongoDB for database operations.
- **Database:** MongoDB (NoSQL)
- **Version Control:** Git

## Usage
To use this application, follow these steps:

1. Clone the repository to your local machine:
   ```sh
   git clone https://github.com/yourusername/Prueba-Full-Stack-Javascript.git
   cd Prueba-Full-Stack-Javascript

2. Install the required dependencies by running:
   ```sh
   npm install

3. Start the development server for both frontend and backend:
   ```sh
   npm start

4. Access the application in your browser at `http://localhost:3000` (frontend) or `http://localhost:5000` (backend).
